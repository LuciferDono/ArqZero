Installation

$ npm i -g arqzero

Quick Start

# Configure your LLM API key
$ arqzero setup
# Start coding
$ arqzero

Headless Mode

Run ArqZero without the interactive TUI for scripting and CI/CD:

$ arqzero -p "fix the bug in src/index.ts"
