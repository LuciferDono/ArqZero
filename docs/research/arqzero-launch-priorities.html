Executive Summary

The AI coding tools market is $8-13B in 2026, growing 20-27% CAGR. 73% of developers use AI tools daily. ArqZero sits in a genuine gap: terminal-native + multi-provider + plugin ecosystem. No competitor has all three. Launch as open-core, price Pro at $15/mo, target HN + r/LocalLLaMA.

Priority 1: Open-Source the Core

What: Push the ArqZero repo to public on GitHub under Apache 2.0.

Why: 57% of developers prefer open-source. GitHub stars are the #1 trust signal. Every day without a public repo is wasted distribution.

Timeline: Day 1.

Steps

  1. Change GitHub repo from private to public
  2. Add LICENSE file (Apache 2.0)
  3. Add CONTRIBUTING.md with contributor onboarding
  4. Add CODE_OF_CONDUCT.md
  5. Add GitHub issue templates (bug report, feature request)
  6. Add .github/FUNDING.yml for GitHub Sponsors
  7. Create "good first issue" labels on 5-10 issues
